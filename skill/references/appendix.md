## 附录一：一页纸快速诊断卡

```
【陈奕迅风格歌词 30秒快速诊断】

□ C3AC检查（结构建筑的灵魂）
  副歌3是否有一个「识穿」句 + 一个「认领」动词？
  □ 识穿句：我知/原来/其实/不过________
  □ 认领动词：唔抆/由佢/由得/当系/等/留/继续/唔畀________
  → 缺一不可。缺认领扣10分起。

□ NSRQ检查（Bridge的呼吸）
  Bridge内部是否有至少3个标注了秒数的[pause]？
  □ 第一口气（0.5秒）□ 画面悬停（2秒）□ 揭穿前（3秒）□ 收尾（1-1.5秒）
  → 少于3个扣5分起。

□ TSMI检查（Hook尾字声调）
  慢歌/歌剧：Hook尾字是否阴上(2)或阳平(4)或阴平(1)？→ 否=换词或接受10分起扣。
  快歌/电子：Hook尾字是否入声(7/8)或阴去(3)？→ 否=换词或接受5分起扣。
  摇滚：随意，靠嘶吼补偿。

□ PAC检查（物象寄生）
  动作链中是否有至少一个「时间寄生」节点？
  □ 物象的某个动作触发了「你以前________」的记忆画面。
  → 无时间寄生扣3-5分。

□ 被动发现检查（主歌叙事）
  主歌中是否有一个「被动发现」瞬间？
  □ 熄mon见到自己 / 手插袋摸到单 / 玻璃蒙咗见到倒影 / 超载灯着自己着咗
  → 全是「我谂起」式主动反思=扣5分。
```

## 附录五：粤语口语核心词库

### 动词（暴力/精准）
㩒（按）、搣（撕）、攥（紧握）、抆（翻找）、揿（按压）、闩（关）、捽（擦）、抰（晃）、搁（放）、揸（握）、揽（抱）、扯（拉）

### 口语助词/虚词
咗（了）、啲（些）、嚟（来）、佢（他/她/它）、冇（没有）、喺（在）、畀（给）、𠮶（那）、嘅（的）、噉（这样）、唔（不）

### 口语短语
专登（故意）、听真啲（听清楚点）、收皮（收工/算了）、求其（随便）、由佢（随他去）、由得（任它）、当系（当作是）

### 「认领」动词（副歌3专用）
留低、唔抆、等、替、当系、由佢、由得、继续、揸住、揽实、唔畀

### 「识穿」词（副歌3专用）
我知、原来、其实、不过、只系、明明
