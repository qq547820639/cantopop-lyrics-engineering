# 高阶法则深挖

## 法则一：寄生动作链三形态

物象的动作链必须在至少一个节点上「寄生」到人的身体感知或情感记忆。

| 形态 | 定义 | 案例 | 寄生效果 |
|------|------|------|----------|
| **触感寄生** | 物象的温度/质感传递到皮肤 | 《31》「佢嘅胶仲系涩嘅」——涩胶质感传到手指 | 人成为物的感知者 |
| **空间寄生** | 物象占据的位置对应人身体缺失部位 | 《超载》「系咪有啲嘢企喺我侧边」——空位对应不存在的陪伴 | 物占据人的边界 |
| **时间寄生** | 物象的动作触发人的记忆时间线 | 《天桥》「佢拉嘅每一个音我都听过」——琴声触发每晚路过的记忆 | 物的时间线覆盖人的时间线 |

**寄生动作链检查清单**：
- [ ] 动作链中是否有触感词（冻/暖/硬/软/涩/湿/干）且与人接触？
- [ ] 物象的位置是否与人身体位置形成对应？
- [ ] 物的至少一个动作是否与人过去的某个动作形成时间叠影？

## 法则二：负空间呼吸量模板（慢歌/歌剧）

Bridge内的沉默必须遵循「短→长→最长」的呼吸递进，模拟人哽咽时的呼吸节律。

| 段落位置 | 呼吸量 | 功能 | 标注方式 |
|----------|--------|------|----------|
| 骤停后第一句 | 0.5秒 | 裸声进入，像突然被推上台 | `[pause— 0.5 sec]` |
| 画面陈述后 | 1-2秒 | 画面在空气中悬停，听众被迫观看 | `[pause—— 2 sec]` |
| 揭穿真相前 | 2-3秒 | 最长的沉默，让听众自己先猜到 | `[pause——— 3 sec]` |
| 真相念白 | 无停顿 | 一口气说完，不给人喘息 | 不加标注 |
| 念白后 | 1-1.5秒 | 收尾呼吸 | `[pause— 1.5 sec]` |

## 法则三：识穿→认领双阶弧线

副歌3的「升华」必须完成从「我被它折磨」到「我需要它」的认知跃迁。

### 第一阶：识穿（揭破幻觉）
- 副歌1-2建立物象与情感的联系
- 副歌3首句应「识穿」这个联系的虚构性

### 第二阶：认领（主动选择幻觉）
- 识穿之后不是「所以我放下了」，而是「所以我选择继续信」
- 用行动动词收尾：「唔抆出嚟」「唔熄mon」「等佢嘟」「由得」「当系」

**「认领」动词词库**：留低 / 唔抆 / 等 / 替 / 当系 / 由佢 / 由得 / 继续 / 揸住 / 揽实 / 唔畀

**识穿词库**：我知 / 原来 / 其实 / 不过 / 只系 / 明明
